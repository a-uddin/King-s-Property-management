# King-s-Property-management
The property management application built on MERN to Manage properties, assets, maintenance, and teams — all in one platform.

A full-stack cloud-native MERN application built to streamline property, asset, and maintenance management across role-based users: Admin, Staff, and External Companies. Designed with scalability, data integrity, and enterprise-level security in mind.

<hr>

<h3>🚀 Key Features</h3>

<ul>
  <li>
    <strong>🔐 Role-Based Access Control</strong><br>
    Custom dashboards for Admin, Staff, and External Company users with isolated permissions and UI.
  </li>
  <li>
    <strong>🧾 Asset & Maintenance Tracking</strong><br>
    Register assets, assign staff for assessment or maintenance, track tasks, and generate reports.
  </li>
  <li>
    <strong>☁️ Cloud-Based Storage & File Uploads</strong><br>
    Invoices are securely uploaded to Amazon S3 using AWS SDK, with secure keys managed via <code>.env</code>.
  </li>
  <li>
    <strong>🔧 Admin Functionalities</strong>
    <ul>
      <li>Manage users and roles</li>
      <li>Approve tasks and assessments</li>
      <li>View maintenance schedules</li>
      <li>Process invoices securely</li>
    </ul>
  </li>
  <li>
    <strong>📊 Staff & External Company Dashboards</strong>
    <ul>
      <li>Update task status</li>
      <li>Submit assessments or invoices</li>
      <li>Access invoice history</li>
    </ul>
  </li>
</ul>
